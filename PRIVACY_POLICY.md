# Privacy Policy for WDYWFM Chrome Extension

**Last Updated:** February 9, 2026

## Overview

WDYWFM is a Chrome DevTools extension that provides network request analysis with session tracking and bot detection. This privacy policy explains how the extension handles data.

## Data Collection and Usage

### What Data We Access

The extension requests the following permissions:

- **storage**: To save user preferences and settings locally
- **webRequest**: To monitor and analyze network requests in the DevTools panel
- **cookies**: To display and analyze cookie data in network requests
- **host_permissions (all_urls)**: To analyze network traffic across all websites when DevTools is open

### How We Use Data

All data processing happens **locally on your device**. The extension:

- Monitors network requests when DevTools is open
- Displays request/response headers, payloads, and cookies
- Analyzes session information and detects potential bot behavior
- Stores user preferences locally

### What We DON'T Do

- We **DO NOT** collect any personal information
- We **DO NOT** send any data to external servers
- We **DO NOT** track your browsing history
- We **DO NOT** share any data with third parties
- We **DO NOT** sell any data

## Data Storage

All data is stored locally on your device using Chrome's storage API. No data leaves your computer.

## Third-Party Services

This extension does not use any third-party services or analytics.

## Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the extension's store listing.
