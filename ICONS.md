# Creating Extension Icons

The extension requires three icon files:
- `icons/icon16.png` (16x16 pixels)
- `icons/icon48.png` (48x48 pixels)
- `icons/icon128.png` (128x128 pixels)

## Option 1: Use the HTML Generator

1. Open `create-icons.html` in your browser
2. Right-click each canvas
3. Select "Save image as..." and save as:
   - `icons/icon16.png`
   - `icons/icon48.png`
   - `icons/icon128.png`

## Option 2: Create Manually

Create simple PNG icons with a network/connection theme. The icons should have:
- Blue background (#1976d2)
- White network nodes and connections
- Simple, recognizable design

## Option 3: Use Online Tools

You can use online icon generators or create simple icons using:
- [Favicon Generator](https://favicon.io/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- Any image editor (GIMP, Photoshop, etc.)

## Note

The extension will work without custom icons - Chrome will use a default icon. However, custom icons improve the user experience.
